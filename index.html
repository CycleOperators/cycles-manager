Index of modules

  • CyclesManager

    A library meant to allow a single canister to act as a cycles battery for receiving requests from and topping up child canisters

  • CyclesRequester

    The CyclesRequester module is meant to be used by canisters that need to request cycles from a battery canister that implements the CyclesManager interface.

  • InterfaceSpec

    Subset of the Interface Spec used for sending cycles For the full Interface Spec, see https://github.com/dfinity/interface-spec/blob/master/spec/ic.did

  • Internal

    DO NOT USE - these are internal types that may change and are not meant to be used by the developer
